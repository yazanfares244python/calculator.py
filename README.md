# calculator.py
What is this project about?

This project is about a calculator that use OOP in a very functional way by using different classes for each job like a class that is responsible for calculations, history data saving and CLI inputs

How to use it or set it up?

When you start it you will get asked for 2 inputs to make numbers and then it gives you menu options and gives you results it will save data if you exit by using option 17 instead of exiting of the terminal directly

What are the features of this project?

The features of this project is that it uses clean OOP structures, has a digit limit of 50000 to make huge calculations while keeping the program safe, saves data through JSON structures for history operations, has options to either enter new numbers or use the previous answer with a new number and uses a clean exit structure and has ID's as identifiers so it will be easier for the user to delete any operation they want
